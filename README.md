# Ritik Sharma — Portfolio

Personal portfolio website of **Ritik Sharma** — AI Engineer & CS student.
Built with **React + Vite + Framer Motion**, featuring a minimal, animated, dark UI.

## ✨ Features
- Smooth Framer Motion animations (preloader, word reveals, magnetic buttons, custom cursor)
- Sections: Hero, About & Journey, Projects, Skills & Achievements, Contact
- Fully responsive, accessible, fast (~95 kB gzipped)

## 🛠️ Tech Stack
React · Vite · Framer Motion · Vanilla CSS

## 🚀 Run locally
```bash
npm install
npm run dev      # start dev server (http://localhost:5173)
npm run build    # production build → dist/
npm run preview  # preview the production build
```

## 📝 Edit content
All profile content lives in [`src/data.js`](src/data.js) — edit it to update text, projects, skills, etc.
